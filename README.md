# TeBuDi (Tempat Buku Digital) 📚

**TeBuDi** adalah platform aplikasi berlangganan buku digital. Proyek ini dikembangkan untuk memenuhi tugas Praktikum Pengembangan Perangkat Lunak Berorientasi Obyek (PPPLBO) Program Studi Teknik Informatika, Universitas Sriwijaya.

Aplikasi ini menggunakan arsitektur *decoupled*, memisahkan antara antarmuka pengguna (Frontend) dan logika server (Backend) untuk skalabilitas dan kemudahan pengembangan.

## 🚀 Fitur Utama
* **Autentikasi Pengguna:** Registrasi dan Login akun (MEMBER/ADMIN).
* **Manajemen Langganan:** Akses ke koleksi buku digital eksklusif.
* **Katalog Buku:** Pencarian dan penjelajahan perpustakaan digital.
* **Keamanan Data:** Enkripsi kata sandi dan proteksi rute API.

## 🛠️ Teknologi yang Digunakan
**Frontend:**
* React.js (dibangun menggunakan Vite)

**Backend:**
* Java 25
* Spring Boot 4.0.3
* Spring Data JPA & Hibernate
* Maven Wrapper (`mvnw`)

---

## ⚙️ Persyaratan Sistem (Prerequisites)
Sebelum menjalankan aplikasi, pastikan kamu sudah menginstal:
* [Node.js](https://nodejs.org/) (versi 18 atau terbaru)
* [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) (versi 25)
* Database MySQL
